PERINTAH SQL (Structured Query Language)




A. Perintah DDL (Data Definition Language)

DDL adalah singkatan dari Data Definition Language yaitu kumpulan perintah pada SQL untuk menggambarkan desain dari database secara menyeluruh, selain itu DDL (Data Definition Language) juga digunakan untuk membuat, merubah maupun menghapus struktur atau definisi tipe data dari obyek yang ada pada database.

Berikut adalah perintah-perintah pada DDL (Data Definition Language) :

1. CREATE

Perintah ini digunakan untuk membuat, termasuk di antaranya membuat database baru, tabel baru, view baru, dan kolom.

Contoh :

CREATE TABLE Buku_Perpus( id INTEGER PRIMARY KEY, kode_buku VARCHAR(20) NULL, judul_buku VARCHAR(225) NOT NULL, tanggal_terbit DATE NULL );

Penjelasan :

Pada contoh perintah di atas memiliki arti bahwa akan di buat suatu table baru bernama Buku_Perpus yang memiliki kolom entitas seperti id yang di beri primary key atau tidak bisa sama dengan data lainya, di sana akan di buat juga kolom Kode_buku dengan tipe data varchar yang memiliki kapasitas tampung sebanyak 20 karakter dan dapat di kosongkan dengan mencatumkan kode NULL, Selain itu ada juga Judul_buku yang bertipe data Varchar dengan kapasitas tampung 225 karakter dan tidak boleh kosong dengan kode NOT NULL. Terakhir ada tanggal_terbit dengan tipe data DATE dan boleh di kosongkan dengan kode NULL.


2. ALTER

Perintah ini digunakan untuk mengubah struktur tabel yang telah dibuat. Pekerjaannya mencakup mengganti nama tabel, menambah kolom, mengubah kolom, menghapus kolom, maupun memberikan atribut pada kolom.

Contoh :

Alter TABLE Buku ADD Penulis Varchar(100);

Penjelasan :

Perintah ini digunakan untuk mengubah struktur table, dimana akan di tambahkan sebuah kolom baru dengan nama Penulis dengan tipe data VARCHAR yang memiliki kapasitas 100 karakter dan akan di tambahkan pada table Buku.


3. DROP

Perintah ini digunakan untuk menghapus database dan tabel.

Contoh :

DROP TABLE Buku;

Penjelasan :

Perintah ini berfungsi untuk menghapus, di sana di gunakan perintah utama DROP yang artinya menghilangkan, setelah perintah DROP di ikuti perintah TABLE yang artinya objek yang akan di hilangkan adalah TABLE, kemudian di ikuti dengan nama objeknya dengan jelas yaitu buku, sehingga perintahnya di baca Menghapus Table Buku.


4. RENAME

Perintah yang digunakan untuk mengubah nama tabel.

Contoh :

RENAME Buku TO Komik;

Penjelasan :

Pada contoh di atas artinya mengubah nama tabel Buku menjadi Komik


5. TRUNCATE

Perintah yang digunakan untuk menghapus semua baris/record dalam tabel dan melepaskan memori yang digunakan oleh tabel.

Contoh :

TRUNCATE TABLE Buku;

Penjelasan :

Pada contoh di atas artinya Buku adalah nama tabel yang akan dihapus seluruh datanya.
